# Happy House

ssafy 1학기 최종프로젝트

<br><br>



## **개요**

- 서비스 페이지 : 부동산 매물 판매 등록 및 관심 매물 등록, 매물 상세보기, 매물 검색
- 관리자 페이지 : 공지사항 및 허위 매물 신고 

<br><br>

## 프로젝트 기간

- 2022년 11월 17일 ~ 2022년 11월 25일
